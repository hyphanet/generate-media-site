To create your own site on Linux:
  • install pyFreenet3 via pip3 install --user pyFreenet3
  • Download gms.zip and extract it somewhere
  • For requirements, see gms.scm
  • add path/to/gms/site to freesitemgr using freesitemgr add with the name gms (edit path/to/site/gms to change the name)
  • Adjust the /?newbookmark link in template.html to point to your siteâ€™s USK
  • Install tor, torify, and youtube-dl
  • adjust attribution if needed
  • Run tor, then call path/to/site/gms URL to add a video or audio
